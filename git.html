Git Commands

  • git init : Used to initialize the git in the project.
  • git add filename: used to add file in the repositery
    • Syntax: git add index.html
  • git status : to check the status of the repositery.
  • git commit
    • Syntax: git commit -m 'Intro to git'
  • git add . : used to add all the files in the repositery
  • git log: used to see or check all the logs or chenges made in repositery
  • git reset --hard and past the log which are going to change.
    • it will remove all the last changes that are made
    • git reset --hard 3e4a9aef937cf24733045bec910864478b33b0f0
  • git reset --soft and paste the log
    • it will moved to the previous version without changing the code
    • git reset --soft 3e4a9aef937cf24733045bec910864478b33b0f0
  • git remote add origin https://git@github.com/[username]/[repository-name].git : to set the origin on github where the repository will added
    • Syntax: git remote add origin https://github.com/ashucj17/vcs.git
  • git branch -M main: making the main branch
  • git push -u origin main: Pushing the code to the main origin branch "in case this code will not work use commands
    • git pull --rebase origin main
    • git push origin main
  • git clone 'project link from git'
    • git clone https://github.com/ashucj17/gitandhub.git
  • git pull: used to pull all the files from the github to local machine
